# ToDoListWebApp
Projet web ASP.NET Core MVC en C# pour gérer une liste de tâches.

Ce projet est en cours de réalisation.

## Lancer le projet en local

1. Copier le fichier :
   appsettings.Development.json.example
   vers
   appsettings.Development.json

2. Adapter le nom de la base de données si nécessaire

3. Lancer les migrations :
   'Update-Database' dans la console du gestionnaire de package
